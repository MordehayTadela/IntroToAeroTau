## Rankine Half-Body Flow Analysis

The Rankine half-body flow is obtained by superimposing a uniform freestream with a point source. This creates a streamlined body with a stagnation point, commonly used to model flow around blunt bodies.

### Key Concepts:
- Superposition of flows
- Source flow behavior
- Stagnation point formation
- Stream function and velocity field
